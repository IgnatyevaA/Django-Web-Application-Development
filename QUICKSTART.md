# Быстрый старт

## Первый запуск

1. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Выполните миграции:**
   ```bash
   python manage.py migrate
   ```

3. **Создайте суперпользователя (опционально):**
   ```bash
   python manage.py createsuperuser
   ```

4. **Запустите сервер:**
   ```bash
   python manage.py runserver
   ```

5. **Откройте в браузере:**
   - Главная страница: http://127.0.0.1:8000/
   - Админ-панель: http://127.0.0.1:8000/admin/

## Основные действия

### 1. Создание получателя
- Перейдите в раздел "Получатели"
- Нажмите "Добавить получателя"
- Заполните: Email, Ф.И.О., Комментарий (опционально)

### 2. Создание сообщения
- Перейдите в раздел "Сообщения"
- Нажмите "Создать сообщение"
- Заполните: Тема письма, Тело письма

### 3. Создание рассылки
- Перейдите в раздел "Рассылки"
- Нажмите "Создать рассылку"
- Выберите:
  - Дата и время начала/окончания
  - Статус (обычно "Создана")
  - Сообщение
  - Получателей (можно выбрать несколько, удерживая Ctrl)

### 4. Отправка рассылки

**Через веб-интерфейс:**
- Откройте детали рассылки
- Нажмите "Отправить рассылку"

**Через командную строку:**
```bash
python manage.py send_mailings
```

## Примечания

- По умолчанию почта выводится в консоль (для разработки)
- Для реальной отправки настройте SMTP в `mailing_service/settings.py`
- Все попытки отправки логируются в разделе "Попытки рассылки"
