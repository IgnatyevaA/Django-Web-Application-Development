### Команды для запуска

1. **Установка зависимостей:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Создание файла .env:**
   Скопируйте `.env.example` в `.env` и заполните значения

3. **Применение миграций:**
   ```bash
   python manage.py migrate
   ```

4. **Создание группы менеджеров:**
   ```bash
   python manage.py create_managers_group
   ```

5. **Создание суперпользователя:**
   ```bash
   python manage.py createsuperuser
   ```

6. **Запуск сервера:**
   ```bash
   python manage.py runserver
   ```

### Примечания

1. **Метод form_valid()**: В проекте используются function-based views с `form.save(commit=False)` для установки поля `owner`. Это эквивалентно методу `form_valid()` в class-based views.

2. **Линтер Flake8**: Большинство ошибок линтера исправлены. Остались только предупреждения о пробелах в пустых строках в файлах команд управления, что не критично.

3. **GitHub**: Проект готов к выкладыванию на GitHub. Убедитесь, что файл `.env` добавлен в `.gitignore`.
